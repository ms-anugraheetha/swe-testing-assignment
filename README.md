# \# Quick-Calc

# 

# Quick-Calc is a simple calculator application built using Python and Tkinter.  

# It supports addition, subtraction, multiplication, division, and clear functionality.  

# The project demonstrates software engineering best practices including unit testing, integration testing, and version control using GitHub.

# 

# \## Features

# 

# \- Addition

# \- Subtraction

# \- Multiplication

# \- Division

# \- Clear function

# \- Backspace button

# \- Graphical User Interface (Tkinter)

# 

# \## Installation

# 

# Install pytest:

# 

# pip install pytest

# 

# \## Run Application

# 

# python gui.py

# 

# \## Run Tests

# 

# python -m pytest

# 

# \## Testing Framework Research

# 

# This project uses pytest. Two common Python testing frameworks are unittest and pytest.

# 

# unittest is Pythonâ€™s built-in testing framework. It is stable and widely used but requires more boilerplate code and is less readable.

# 

# pytest is more modern and easier to use. It requires less code, has better output, and automatically discovers tests. It also supports advanced features like fixtures and parameterization.

# 

# pytest was chosen because it is simpler, faster, and more suitable for small and medium projects.
